Entrainement d'un véhicule sur une route simple

Dans cette situation, on entraine notre agent sur une route sans véhicules à une voie. Son reward est construit en suivant les points suivants :
- Vitesse (++)
- Accélération (-)
- véhicule centré (--)
- sortie de route (---)
